# Project overview

Last updated: 2026-07-20

Creator Match discovers Indian micro-influencers from live public evidence, verifies usable platform signals, and helps authenticated users prepare carefully reviewed outreach. The assignment-grade target is up to 50 creators, but the system must report whatever live evidence actually yields.

Anonymous visitors receive an ephemeral preview of up to eight results for one niche. Signed-in users can request a larger live shortlist, save creators, download current results, generate constrained email and DM drafts, and confirm a Gmail send. There is no creator seed dataset.

The product supports Instagram and YouTube across Fashion, Beauty, Fitness, Food, Finance, Parenting, Tech, Gaming, and Education. The UI is an editorial dark experience with Outfit typography, evidence-forward messaging, a gapless bento section, marquee, GSAP reveal/pinning/stacking, and explicit shortfall states.
